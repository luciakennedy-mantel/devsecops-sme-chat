## Quick Setup

1. **Clone and install**
   ```bash
   git clone https://github.com/luciakennedy-mantel/devsecops-sme-chat.git
   cd devsecops-sme-chat
   npm install
   ```

2. **Add your API key**
   ```bash
   cp .env.example .env
   # Edit .env and add your OpenAI API key
   ```

3. **Start the chat**
   ```bash
   cd ../Cline/MCP/devsecops-docs-server
   npm install
   ./start-chat.sh
   ```
   
   Access at: http://localhost:3001
